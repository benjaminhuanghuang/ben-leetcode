
## Complexity
- O(1) or O(C): constant time
- O(log2N)    : sub-linear time
- O(N)        : linear time
- O(Nlog2N)   : 
- O(N^2)      : quadratic time
- O(N^3)      : cubic time
- O(2^N)      : exponential time
